# How to run ? 



```
Windows
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

```
Linux + macOs
pip3 install -r requirements.txt
python3 manage.py migrate
python3 manage.py runserver
```

Navigate to to http://127.0.0.1:8000
